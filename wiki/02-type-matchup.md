## 「迷ったらこう考える」判断ルール

**ルール1: まず「無効」を確認する（相手が完全に無駄にならないために）**

```
相手がじめんタイプ → でんき技は無効。みず/くさ/こおりを使え
相手がノーマルタイプ → ゴースト技は無効。かくとう技を使え
相手があくタイプ → エスパー技は無効。かくとう/フェアリーを使え
相手がドラゴンタイプ → フェアリー技は2倍。氷技も2倍
相手がひこうタイプ → じめん技は無効。いわ/でんき/こおりを使え
```

**ルール2: はがね・フェアリー複合は「最強耐性」**

はがね/フェアリー（ザシアン等）の弱点はほのおとじめんだけです。
ドラゴン・どくを完全無効、フェアリーは1/4耐性（はがね×フェアリー双方が半減するため）。
弱点2（ほのお・じめん）・無効2（どく・ドラゴン）の圧倒的な耐性構成です。

> **異なる見方:** はがね/フェアリー複合は確かに耐性が優秀ですが、「弱点がほのお・じめんの2つしかない」ということは、環境にほのお・じめん技が豊富に存在する場合には想定より刺さります。「耐性が優秀＝入れるだけで強い」と単純には言えず、パーティ全体でほのお・じめんへの対策が必要です。

**ルール3: 4倍弱点を狙う**

> **注:** 以下は「入門用の単純化モデル」です。実際の環境ではハイブリッド型構築が主流であり、単純に4倍弱点を狙えば勝てるわけではありません。

```
くさ/どく複合（例: フシギバナ） → じめん技が4倍（くさ×2・どく×2）
  ※ほのおはむしろ半減（0.5倍）。エスパーは2倍弱点（4倍ではない）
ほのお/ひこう複合（例: ファイアロー・ファイヤー） → いわ技が2倍
  ※4倍ではなく2倍（ほのおへのいわは等倍、ひこうへのいわが2倍）
いわ/ひこう複合（例: アーケオス） → でんき・みず・くさが4倍
```

**ルール4: 複合タイプの耐性を把握する優先順位**

対戦で出会う頻度が高い順に把握します。まずこれだけ：

- じめん/ひこう（ランドロス等）：でんき・じめんを両方無効化
- みず/じめん（ヌオー等）：でんきを完全無効（てんねん特性と相まって強い）
- ドラゴン/はがね：どくが無効、弱点がかくとう/じめんの2つだけ

## タイプ相性の世代変遷（過去記事を読む際に必須）

| 世代 | 変更内容 | 影響 |
|------|----------|------|
| **第1世代** | ゴースト技→エスパー：プログラムバグで「無効」に（本来は抜群のはずだった）。当時はあく・はがねタイプが存在しなかったことも一因 | エスパー無双の遠因 |
| **第1世代** | むし技→どく：抜群（現在は等倍）/ どく技→むし：抜群（現在はいまひとつ） | 現在と真逆の相性 |
| **第2世代** | あく・はがねタイプ追加（15→17タイプ）。ゴースト→エスパーが正常化されて抜群に | エスパー無双終焉 |
| **第6世代** | フェアリータイプ追加（17→18タイプ） | ドラゴン全盛終焉 |
| **第6世代** | ゴースト・あく技→はがね：いまひとつ→**等倍**に緩和 | はがねタイプの大幅弱体化 |
