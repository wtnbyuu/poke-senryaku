# 3. 育成チェックリスト

> [前のページ](02-type-matchup.md) | [目次](README.md) | [次のページ](04-damage-calc.md)

**このページを読んだらできるようになること**
→ 種族値の読み方から性格・努力値・個体値まで、何をどの順番でやればいいか分かります。迷わず育成を完成できます。

---

## ステップ1: 種族値を確認する

**確認場所:** ポケモン徹底攻略（yakkun.com）でポケモン名検索 → 「種族値」欄を見る

**読み方チェックリスト**

- [ ] H（HP）・A（攻撃）・B（防御）・C（特攻）・D（特防）・S（素早さ）の6項目を確認
- [ ] Aが高い → 物理技メインが基本。性格は「いじっぱり」か「ようき」が多い
- [ ] Cが高い → 特殊技メインが基本。性格は「ひかえめ」か「おくびょう」が多い
- [ ] AとC両方高い → 両刀型。性格は使う技の比重に応じて決める
- [ ] Sが100以上 → 素早さ重視で考える。110以上はスカーフ採用の相手に注意
- [ ] 合計種族値が600以上 → 「600族」。禁止伝説でなくとも高スペック

**素早さ種族値と実戦ライン（Lv50・個体値31・努力値252）**

| 種族値 | 代表ポケモン | 最速（S252・性格補正↑） | 準速（S252・補正なし） |
|---|---|---|---|
| 80 | カイリュー | 145 | 132 |
| 84 | サーフゴー | 149 | 136 |
| 111 | ウーラオス | 179 | 163 |
| 130 | コライドン・ミライドン | 200 | 182 |
| 135 | ハバタクカミ・パオジアン | 205 | 187 |
| 136 | テツノツツミ | 206 | 188 |

こだわりスカーフ（使用者のタイプ一致技しか出せなくなる代わりに素早さ1.5倍の持ち物）装備でS実数値×1.5（端数切り捨て）。スカーフカイリュー最速（145×1.5≒217）はハバタクカミ最速（205）を抜けます。

## ステップ2: 性格を決める

**アタッカー向け性格（4択から選ぶだけ）**

| 性格 | 上昇 | 下降 | 選ぶ場面 |
|---|---|---|---|
| いじっぱり | 攻撃+10% | 特攻-10% | 物理技メイン・素早さは気にしない |
| ようき | 素早さ+10% | 特攻-10% | 物理技メイン・速さも必要 |
| ひかえめ | 特攻+10% | 攻撃-10% | 特殊技メイン・素早さは気にしない |
| おくびょう | 素早さ+10% | 攻撃-10% | 特殊技メイン・速さも必要 |

**耐久型向け性格**

| 性格 | 効果 |
|---|---|
| わんぱく / ずぶとい | 防御+10%（物理受け） |
| しんちょう / おだやか | 特防+10%（特殊受け） |

> HPには性格補正がかかりません。

**チャンピオンズの場合:** 「能力適性（Stat Alignment）」という名前に変わりましたが、機能は同じです。VP（ゲーム内通貨）を使えばいつでも変更可能です。

## ステップ3: 努力値を振る

**基本の3パターン（迷ったらこれ）**

```
物理アタッカー: 攻撃252 / 素早さ252 / HP4
特殊アタッカー: 特攻252 / 素早さ252 / HP4
物理受け: HP252 / 防御252 / 特防4
```

**確定2発・確定耐えを目指す場合**

ダメージ計算機を使います。「確定耐えするために防御にあと何振り必要か」を計算し、
余った努力値をHPに回します。[4章 ダメージ計算](04-damage-calc.md)の手順を先に読んでも良いです。

**HP調整の基本ルール**

- HP実数値は**奇数推奨**：砂嵐・宿り木・いのちのたまの定数ダメ1点を節約できる
- 食べ残しを持たせるなら **HP実数値を16の倍数+1（16n+1）** に近づける：回収効率が最大化
- 砂嵐やけど環境では **HP実数値を16n-1** に：1/16スリップダメージを最小化（努力値を「減らして」調整するのが正しい）

**努力値の上限**

- SV（第3世代以降の共通仕様）：1能力に最大252。合計510まで
  ※第6世代（XY）で旧255上限が252に統一されたが、252・510の基本概念は第3世代から存在する
- チャンピオンズ：「能力ポイント（SP）」という新システムに変更。合計66ポイント、1能力最大32ポイントで、2能力に32ずつ振っても2ポイント余る。実数値に直接加算される新方式（詳細は公式サイトを参照）

## ステップ4: 個体値を確認・調整する

**チャンピオンズは不要:** 個体値が廃止され全31固定。このステップはSV専用です。

**SVでの最低チェックリスト**

- [ ] メインで使う能力（攻撃 or 特攻）：V（31）確認
- [ ] 素早さ：V確認
- [ ] HP：用途に応じて確認（奇数調整が必要な場合は特に）
- [ ] 使わない能力（物理アタッカーならとくこう等）：0推奨（イカサマ対策等）

**SV: 最短育成ルート**

1. 5Vメタモンを星6テラレイドで入手（または交換）
2. あかいいとで5V卵を生産
3. ミントで性格補正を変更（ラッキーズで20,000円）
4. ドーピングアイテムで努力値を252まで振る（各ステータス上限まで26,000円相当）
