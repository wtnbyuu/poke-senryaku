## 30秒で判断する方法：ダメージ計算機を使う

**使うツール:** ポケソル（https://pokesol.com）または Pokemon Showdown の計算機

**手順（30秒で完了）**

```
1. 「攻撃側ポケモン」を入力（努力値・性格・持ち物を設定）
2. 「防御側ポケモン」を入力（HP・防御/特防を設定）
3. 「技名」を選んで計算
4. 「最小〜最大ダメージ」と「確定数」を読む
```

**計算結果の読み方チェックリスト**

| 用語 | 定義 | 確率 | 実戦的な扱い |
|---|---|---|---|
| **確定1発（確1）** | 最小乱数0.85でもHP100%以上 | 100% | 必ず倒せる |
| **高乱数1発（高乱1）** | 16段階中15以上で1発 | 93.75% | ほぼ確実・期待してよい |
| **乱数1発（乱1）** | 最大乱数1.00のみで1発 | 6.25% | **前提にしない** |
| **確定2発（確2）** | 最小乱数×2で確実に倒せる | 100% | 2ターンで倒せる |
| **確定耐え** | 最大乱数でも耐える | 100% | 必ず耐える |

乱数は0.85〜1.00の16段階で毎回発生します。最小と最大の差は15%です。

## ダメージ計算式（理解のために）

```
ダメージ = floor(floor(floor(floor((Lv×2÷5+2) × 威力 × 攻撃÷防御) ÷50) + 2) × 各補正)
```

> Lv=ポケモンのレベル、威力=技の基本威力、攻撃=攻撃側の攻撃/特攻実数値、防御=防御側の防御/特防実数値

計算機を使えばよいですが「なぜその数値になるか」を理解するための公式です。端数処理（floor=切り捨て）は4回行われ、処理の順序によって結果が変わります。

**主要補正の倍率早見表**

| 補正 | 倍率 | 補足 |
|---|---|---|
| タイプ一致（STAB） | ×1.5 | STAB（Same Type Attack Bonus）= 使用ポケモンのタイプと技タイプが一致するときの威力補正。五捨五超入で端数処理 |
| タイプ一致（テラスタル同タイプ） | ×2.0 | 強化STAB |
| こだわりハチマキ / メガネ | ×1.5 | 攻撃力または特攻力に乗る |
| いのちのたま | ×約1.3（正確には5324/4096。計算機推奨） | 最終ダメージに乗る |
| 晴れ/雨での一致技 | ×1.5 | 天候ブースト |
| 抜群（2倍弱点） | ×2.0 | |
| 4倍弱点 | ×4.0 | |

## 実戦でよくある計算シナリオ

**「1発で落とせるか？」** — 攻撃側・防御側・技名を入力 → 確定数を読む。「確定2発（50〜60%）」なら1発で落ちないので反撃を想定して動きます。

**「この攻撃を耐えられるか？」** — 相手の持ち物・補正込みで計算。「確定耐え」なら突っ張れます。「高乱数耐え」なら守るを検討します。

**「相手の持ち物を特定する」** — 実際に食らったダメージから逆算できます。CS特化ハバカミのシャドーボールが「確定2発」のはずなのに「確定1発」だったらメガネかスカーフ。技が固定されていたらスカーフの可能性が高いです。
