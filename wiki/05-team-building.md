## 4ステップ構築法

**Step 1: 軸を決める**

「この2体で勝ちに行く」という核を先に決めます。

```
条件:
- 2体でお互いの苦手タイプをカバーできる
- 2体合わせて「攻め」と「受け」が成立する
- この並びで勝てるシナリオが具体的に描ける
```

**軸の例**

| 組み合わせ | 役割分担 |
|---|---|
| カイリュー + サーフゴー | カイリュー（物理エース）+ サーフゴー（特殊崩し・変化技無効） |
| ハバタクカミ + ランドロス | 特殊エース + 物理崩し・威嚇クッション |
| 積みエース + 起点作り | 壁/あくび/ステロでエースを動かす構成 |

**Step 2: 補完を加える**

軸の2体が苦手とするタイプ・構築に対して「最低限対応できる」ように残り4体を埋めます。

```
補完を考えるチェックリスト:
- [ ] 軸2体が共通して弱いタイプは何か？ → そのタイプを半減・無効にできるポケモンを入れる
- [ ] 物理技と特殊技のバランスは取れているか？ → 偏ると1体の受けポケに詰む
- [ ] ステルスロック4倍弱点のポケモン（例: ほのお/ひこう複合など、いわタイプに4倍弱点を持つ）が複数いないか？
- [ ] 軸2体が止まる（起点にされる）ポケモンへの対応はあるか？
```

**Step 3: 崩しを用意する**

「軸2体が通じない相手」への解答を1〜2体入れます。

```
崩し枠の役割:
- 受けループ・高耐久構築を突破する
- 積み技・変化技で有利状況を作って全抜きを狙う
- トリックで相手の受けポケを機能停止させる

代表的な崩し枠:
- わるだくみサーフゴー（変化技無効で積める、サイコショックで物理受けに刺さる）
- 剣の舞セグレイブ（物理最強クラスで受けを突破）
- こだわりスカーフゴチルゼル（かげふみ+トリックで受けポケを縛る）
```

**Step 4: クッションを確保する**

「不利対面をリセットして、有利対面を作り直す」役割を1体入れます。

```
クッション枠の条件:
- とんぼがえり or ボルトチェンジで安全な交代ができる
- 苦手なポケモンを受けつつ次の展開につなげられる

代表的なクッション:
- 霊獣ランドロス（威嚇+とんぼがえり）
- アーマーガア（鉄壁+ボディプレス+はねやすめ）
- カバルドン（ステルスロック+あくびで起点作りも兼任）
```

---

## 構築フォーマット「2-2-1-1」

```
軸2体      : 最多選出。この並びで勝ちを狙う
取り巻き2体 : 軸をサポート・補完する
補完1体    : 苦手構築への解答
遊撃1体    : 複数の役割をこなせる汎用枠
```

**完成判断チェックリスト**

- [ ] 環境上位5構築それぞれに「ゼロ回答にならない」選出ができる
- [ ] 「軸2体」に対して相手が何を出してきても2通り以上の返しがある
- [ ] 物理技と特殊技が構築に両方存在する
- [ ] ステルスロック4倍弱点のポケモンが多くない（2体以内が目安）
- [ ] テラスタル（SVの場合）の使い道が明確な枠が1〜2体ある
