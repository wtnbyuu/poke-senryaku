## 受けループを識別する

**SV環境の典型的な受けループ6体構成（S23 最終129位・R2042 構築例）**

| ポケモン | 役割 | キーとなる技・特性 |
|---|---|---|
| ドヒドイデ | どくびし撒き・積み対策 | くろいきり（積みリセット）・じこさいせい・メンタルハーブで挑発対策 |
| ハピナス | 特殊全受け | 格闘テラスタルで崩し対策・エコーボイスで瞑想対抗 |
| ヘイラッシャ | 物理受け・パオジアン対策 | しろいいし回復・ウェーブタックル |
| アーマーガア | 物理要塞化 | 鉄壁+ゴツゴツメット+ボディプレス+はねやすめ |
| グライオン | 特性ポイズンヒール | じしんorハサミギロチン・まもる |
| サーフゴー | 特殊受け兼アタッカー | 変化技無効（おうごんのからだ） |

**識別チェックリスト**

- [ ] 相手が全く攻撃してこず、「どくどく」「ウィルオウィスプ」「宿り木」を多用している
- [ ] 毎ターン回復技を使っている
- [ ] 交代を繰り返しながらこちらのポケモンの毒/状態異常ダメージを稼いでいる
- [ ] ステルスロック+どくびしが両方展開された

→ 3つ以上当てはまれば受けループを疑います。

## 崩し手段5選（構築段階で1つ以上用意しておく）

**崩し手段フロー**

```
自分のパーティに崩し枠はあるか？
       ↓Yes                         ↓No
どの崩し手段を持っているか確認    →「次から崩し枠を入れる」と記録
       ↓
以下の中で使える手段を選ぶ
```

> **補足:** 以下は代表的な5手段です。高火力技での強引な削り・消耗戦（ステルスロック+高打点で削りきる）なども実戦的な選択肢です。「5選」は網羅ではなく代表例です。

**手段1: 挑発（ちょうはつ）**

```
使い方: 受けポケが変化技を使おうとした瞬間に先読みで挑発
効果: 3ターン変化技を封印
代表: 挑発ハバタクカミ（受けループ使用者が「最も重い」と評価する筆頭）
注意: 相手がメンタルハーブを持っていると1回無効化される → 2回目で再封印
```

**手段2: みがわり + 積み技**

```
使い方: みがわりを置いて変化技を全シャットアウト → 積んで突破
代表: みがわりわるだくみサーフゴー
注意: ほえる・ふきとばし持ちは貫通交代あり
     くろいきりはみがわりを貫通してランクをリセットする（みがわりで防げない）
     → くろいきりを打たれたら積みがリセットされるので、1ターン攻撃or変化技で耐えて再積み直す
```

**手段3: トリック（こだわりアイテム押し付け）**

```
使い方: こだわりスカーフゴチルゼルの「かげふみ」で逃がさず → トリック
効果: 受けポケにスカーフが押し付けられ、変化技の連打が不可能になる
対象: チオンジェン・ヒードラン・ディンルーに特に有効
注意: 相手がなげつけるで返してくる場合あり
```

**手段4: サイコノイズ（回復封じ）**

```
使い方: 2ターン回復技を封じながらダメージを与える
代表: わるだくみマシマシラ（サイコノイズ+みがわり+格闘テラバースト）
効果: グライオンの自己再生を封じて削る → 格闘テラバーストでラッキーも崩せる
```

**手段5: デバフ技（防御ランク低下）**

```
使い方: アシッドボム（特防-2） or ルミナコリジョン（特防-2）
効果: 相手の特防ランクを下げてダメージを通しやすくする
注意: てんねん（クレベース・ハピナス等）はランクの上昇・下降を両方無視する特性のため、
     アシッドボムによる特防ダウンも効果がない。てんねん持ちへの崩しは
     サイコショック（物理防御参照の特殊技）や格闘技など、ランクに依存しない手段を用いること
```

## 受けループ攻略チェックリスト（対戦中に確認する）

- [ ] どくびしを展開されたか確認 → 鋼・毒タイプで無効化、こおり・ひこうタイプで着地回避
- [ ] グライオンへの回答はあるか → サイコノイズ持ち、フェアリー技、または挑発で機能停止
- [ ] ハピナス/ラッキーへの回答はあるか → 格闘技、サイコショック（物理防御参照）
- [ ] くろいきりを打たれたか → 積みがリセット。1ターン耐えてから再積みする
- [ ] テラスタルでタイプ補完された → 両刀型（物理+特殊）で対応する
