## カイリュー（SV・マルチスケイル型）

**代表調整 H244 / B12 / D44 / 残りAS**（性格: しんちょう or ようき）

| 目的 | 達成するライン |
|---|---|
| D44 | 臆病眼鏡ハバタクカミ(C187) テラスムーンフォースをマルチスケイル込みで2耐え |
| B12 | オーガポンへの乱数改善（B4 → B12） |
| H244 | ガチグマ（暁）C特化ブラッドムーンを確定耐え |

**食べ残し竜舞型の場合:** HP実数値を **16n+1** になる振り数に調整して食べ残し回収効率を最大化。

## ディンルー（SV）

特性「わざわいのうつわ」で場に出るだけで相手全員の特攻が×3/4に。

**型1: HB特化**（仮想敵: 物理アタッカー全般）

```
持ち物: オボン or 食べ残し
採用技: ステルスロック + ふきとばし（サイクル系に効果的）
```

**型2: HD特化 + とつげきチョッキ**（仮想敵: 特殊アタッカー全般）

```
持ち物: とつげきチョッキ（変化技使用不可）
仮想敵: ハバタクカミ・イーユイ・ミライドン
耐久ライン: 眼鏡イーユイのテラス炎オーバーヒートを最高乱数以外耐え（高乱数耐え）
```

**使い分け基準**

```
パーティの物理受けが薄い → HB特化 + 変化技あり
パーティに特殊受けが足りない → HD + チョッキ
```

## 歴代の有名な耐久調整（参考）

**201ガブリアス（第4世代・DP）** — HP実数値201。ハチマキヘラクロスのインファイト（物理最大脅威）とライコウのめざめるパワー氷（特殊最大脅威）の両方を「1つの数字」で同時に確定耐えした。205調整版は砂嵐+みがわり型向け（ラッキーの地球投げでみがわりが壊れない）。

**HBクレセリア（第4〜6世代）** — 図太い H252/B252/S4、実数値H227/B189、持ち物ゴツゴツメット（PGLデータ採用率93.6%）。HB特化でメガボーマンダの捨て身タックルを2耐え、特化ガブリアスの逆鱗を2耐え。

## 耐久調整の基本原則（応用するとき）

**HP調整の優先ルール**

| 状況 | 調整方針 |
|---|---|
| 食べ残し採用 | HP実数値を **16n+1** に近づける（回収効率最大） |
| 砂嵐・宿り木環境 | HP実数値を **16n-1** に（毎ターンダメを最小化。努力値を「減らす」方向で調整） |
| 汎用耐久向上 | HP実数値を**奇数**に（定数ダメージを1節約） |

**理想の耐久配分の考え方**

```
物理・特殊を均等に受けたい → H ≒ B + D になるよう配分（H=B+D法則）
物理受けに特化したい → H と B を均等に高める
HPが異常に高いポケモン（ハピナス等）→ BD特化が最適（法則の例外）
```
