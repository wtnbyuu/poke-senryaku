# 10. SV勢のためのチャンピオンズ移行チェックリスト

> [前のページ](09-generations.md) | [目次](README.md) | [次のページ](appendix.md)

**このページを読んだらできるようになること**
→ SVから「ポケモンチャンピオンズ」（2026年4月リリース）に移行する際に「何が変わったか」「何を覚え直すか」が分かります。

> **重要:** 以下の情報はすべて2026年5月時点のものです。チャンピオンズはリリースから間もなく、アップデートにより仕様が変更される可能性があります。公式サイト・公式SNSで最新情報を必ず確認してください。

> **異なる見方:** チャンピオンズはリリースからわずか4週間（2026年5月時点）のタイトルです。特にランクマッチのTOD判定・SPの上限・状態異常の数値はバランスパッチで変更される可能性が高い項目です。本章の数値は「現時点の仕様」として参考程度に扱い、公式サイトで必ず確認してください。

## SVとの主要変更点 比較表

| 要素 | SV | チャンピオンズ（2026年5月時点） |
|---|---|---|
| **個体値（IV）** | 0〜31のランダム厳選必要 | **全31固定（廃止）** |
| **努力値（EV）** | EV（最大252×2、合計510） | **能力ポイント（SP）に変更。合計66ポイント、1能力最大32ポイント** |
| **性格** | 25種類（ミントで変更可） | 「能力適性（Stat Alignment）」に名称変更。機能は同じ。**VP（ゲーム内通貨）課金で自由変更** |
| **ギミック** | テラスタル | **メガシンカが復活（1試合1回、チーム全体で1匹）** |
| **TOD（時間切れ判定）** | あり（残数→HP割合で判定） | **ランクマッチは引き分け**に変更。大会は従来通り |
| **まひ** | 行動不能25% | **行動不能12.5%（半減）** |
| **こおり** | 毎ターン20%で解除 | **3ターン目で確定解除** |
| **ねむり** | 1〜3ターンランダム | **3ターン目確定起き** |
| **育成コスト** | 孵化・厳選作業が必要 | **VP課金で即変更可能** |

## 移行チェックリスト

**「覚え直す」ことリスト**

- [ ] **ギミックがメガシンカになった** → SVのテラスタル戦略ではなくメガシンカを軸に考える
- [ ] **状態異常の仕様が変わった** → まひは12.5%（前:25%）、こおりは3ターン確定解除、ねむりは3ターン確定起き
- [ ] **努力値配分の考え方が変わった** → SP（能力ポイント）で合計66ポイントを配分する新方式
- [ ] **TOD戦術はランクマッチで使えない** → 両者引き分けになる。大会では従来通り
- [ ] **性格変更がゲーム内で自由にできる** → VP消費で変更

**「変わらない」ことリスト（そのまま活かせるSVの知識）**

- [ ] タイプ相性（18タイプ・第6世代以降と同じ）→ [2章 タイプ相性](02-type-matchup.md)
- [ ] ダメージ計算の基本式 → [4章 ダメージ計算](04-damage-calc.md)
- [ ] 構築の考え方（軸・補完・崩し・クッションの概念）→ [5章 構築の組み方](05-team-building.md)
- [ ] 選出論・安定行動の考え方 → [6章 選出の決め方](06-team-selection.md)
- [ ] 受けループ・サイクル・積み構築の基本メカニクス → [7章 受けループ対策](07-stall-breaking.md)
