## よく使う用語クイックリファレンス

| 用語 | 意味 |
|---|---|
| **三値（さんち）** | 種族値・個体値・努力値の3つ |
| **HABCDS** | H=HP, A=攻撃, B=防御, C=特攻, D=特防, S=素早さ |
| **STAB** | Same Type Attack Bonus。使用ポケモンのタイプと技タイプが一致するときの威力×1.5補正 |
| **V（ブイ）** | 個体値31のこと |
| **ぶっぱ** | 1ステータスに努力値252を全振り |
| **最速** | S補正性格+IV31+EV252の最大素早さ |
| **準速** | 補正なし性格+IV31+EV252 |
| **確2** | 2発で確実に倒せる |
| **高乱1** | 16段階中15以上で1発（93.75%） |
| **乱1** | 最大乱数のみで1発（6.25%） |
| **確定耐え** | 最大乱数でも耐える |
| **一貫** | 特定の技・タイプが相手の多くに通る状態 |
| **崩し** | 受けループ等の耐久戦術を突破すること |
| **起点** | 積みエースが積みやすい状況 |
| **TOD** | 時間切れ判定で勝ちを狙う戦術（SVまで。チャンピオンズランクマッチは引き分け） |
| **BO1/BO3** | 1本勝負 / 2本先取の試合形式 |
| **OU/Ubers** | SmogonのTier。Ubersは禁止級、OUは通常環境最上位 |
| **16n-1** | HPを16で割って1引いた値。定数ダメ最小化の調整 |
| **とんボルチェン** | とんぼがえり+ボルトチェンジ。安全に交代しながらダメージも与える |
| **択（たく）** | 相手の行動によって最善手が変わるジャンケン場面 |
| **SP（エスピー）** | チャンピオンズの「能力ポイント」。SVの努力値に相当する新システム |
| **VP（ブイピー）** | チャンピオンズのゲーム内通貨。性格変更・育成変更に使用 |
